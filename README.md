# Mission Status React Demo

Mission Status is a small aerospace-themed React project that simulates basic spacecraft telemetry.

I built this project as a beginner React portfolio demo. The data is simulated, so it does not connect to a real spacecraft or API.

## Features

- Mission timer
- Pause and resume button
- Reset button
- Simulated altitude, speed, battery, and temperature
- Animated spacecraft orbit
- Basic spacecraft system status
- Responsive layout

## Technologies Used

- React
- JavaScript
- CSS
- Vite

## How to Run the Project

Make sure Node.js is installed.

Open the project folder in a terminal and run:

```bash
npm install
npm run dev
```

Vite will display a local link. Open that link in your browser.

## Build the Project

```bash
npm run build
```

## Why I Made This

I made this project to practice React components, props, state, effects, and responsive web design while connecting the project to my interest in aerospace and mission operations.
